Habilidad Susurrador de Bugs para OpenClaw.

## Qué Soluciona
Analiza patrones de error para predecir y prevenir bugs.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque primero en seguridad
- Soporte para rollback

## Uso
/bug-whisperer

## Ejemplo
```
Analizar código para problemas de análisis.
```

## Instalación
Esta habilidad está automáticamente disponible en OpenClaw.
